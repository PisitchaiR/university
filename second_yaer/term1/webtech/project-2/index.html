tailwind คืออะไร??
Tailwind เป็นเฟรมเวิร์ก CSS แรกของยูทิลิตี้ ตรงกันข้ามกับ CSS framework อื่น ๆ เช่น Bootstrap หรือ Materialize CSS ซึ่งไม่ได้มาพร้อมกับส่วนประกอบที่กำหนดไว้ล่วงหน้า แต่ Tailwind CSS ทำงานในระดับที่ต่ำกว่าและมอบชุดของคลาสตัวช่วย CSS ให้คุณ ด้วยการใช้คลาสนี้คุณสามารถสร้างการออกแบบที่กำหนดเองได้อย่างรวดเร็ว Tailwind CSS ไม่ได้ให้ความเห็นและให้คุณสร้างการออกแบบที่เป็นเอกลักษณ์ของคุณเอง
รายละเอียดของ tailwind
CSS framework ที่มาพร้อม style พื้นฐานที่มีให้ตั้งแต่ color palette, spacing, shadow ฯลฯ เตรียมไว้ให้เสร็จสรรพ ข้อดีคือหน้าตาของเว็บเราก็จะไปในแนวทางเดียวกัน มีเฉดสี มีสเกลให้ ทำให้เราควบคุมความหลากหลายของตัวเลขและค่าต่างๆ ไม่ให้ฟุ้งและกระจัดกระจายเกินไป
วิธีการติดตั้ง
  1. โดยการใช้ cdn
    2. <link href="https://unpkg.com/tailwindcss@^2/dist/tailwind.min.css" rel="stylesheet">
  2. ติดดั้งจาก node
    3. 1. สร้างโฟร์เดอร์ แล้วเปิด terminal แล้วเข้าถึงไปยังที่อยู่ของไฟล์นั้น แล้วพิมพ์คำสั่ง
    npm install tailwindcss
    2. สร้างไฟล์ main.css
    @tailwind base;
    @tailwind components;
    @tailwind utilities;
    3. แก้ไขในไฟล์ package.json โดยเพิ่มคำสั่ง
    {
    "name": "projectweb",
    "version": "1.0.0",
    "description": "",
    "main": "index.js",
    "scripts": {
    "test": "echo \"Error: no test specified\" && exit 1",
    "build:css": "tailwindcss -i ./ที่อยู่ของไฟล์ main.css -o ./ที่อยู่ของไฟล์ที่จะสร้าง
    (ตัวอย่าง tailwind.css)"
    },
    "keywords": [],
    "author": "",
    "license": "ISC",
    "dependencies": {
    "tailwindcss": "^2.2.17",
    }
    }
    4. พิมพ์ใน terminal เพื่อาสร้่างไฟล์ tailwind
    npm run build:css
